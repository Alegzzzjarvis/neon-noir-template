02 — Structure

File Structure

The template ships as a single HTML file. For deployment, the recommended structure is:

📁 neon-noir/
index.html  ← main template file
hero-bg.mp4  ← your hero video (optional)
📁 documentation/
index.html  ← this file
INFO

The entire template is self-contained in index.html. The only external dependency is the Three.js CDN and Google Fonts — both load automatically.

03 — Quick Start

Quick Start

01

Open the file

Simply open index.html in any modern browser. No server, no build step, no npm install needed.

02

Edit your content

Open index.html in any text editor (VS Code recommended). Search for the placeholder text and replace it with your own content.

03

Add your video (optional)

Place your video file as hero-bg.mp4 in the same folder. If no video is found, the hero falls back to a beautiful animated gradient.

04

Deploy

Upload to any hosting: GitHub Pages, Netlify, Vercel, or a traditional web host. See the Deployment section for details.

04 — Colors

Colors & Theme

All colors are defined as CSS variables at the top of the <style> block. Change them once and they update everywhere.

/* Find this block near the top of index.html */
:root {
  --neon-cyan:   #00f5ff;  /* primary accent */
  --neon-pink:   #ff0080;  /* secondary accent */
  --neon-purple: #bf00ff;  /* tertiary accent */
  --dark:        #020208;  /* main background */
  --dark2:       #07070f;  /* card backgrounds */
  --dark3:       #0d0d1a;  /* subtle surfaces */
  --text:        #e8e8f0;  /* primary text */
  --text-dim:    #9494b0;  /* secondary text */
}
--neon-cyan
#00f5ff
--neon-pink
#ff0080
--neon-purple
#bf00ff
--dark
#020208
TIP

To change the entire color scheme, just replace --neon-cyan with any color (e.g. #00ff88 for green neon). The gradient accents will update automatically.

05 — Typography

Typography

The template uses three Google Fonts loaded in the <head>:

FontUsageWeight
Bebas NeueHeadlines, titles, large display text400
SyneBody text, descriptions, UI labels400, 700, 800
JetBrains MonoTags, navigation, code-style labels300, 400

Changing Fonts

Find the Google Fonts <link> tag in the <head> and swap the font names:

/* In the <head> */
<link href="https://fonts.googleapis.com/css2?family=Bebas+Neue&...">

/* In CSS, update the font-family references */
font-family: 'Bebas Neue', sans-serif;  /* headlines */
font-family: 'Syne', sans-serif;        /* body */
font-family: 'JetBrains Mono', monospace; /* mono labels */
06 — Hero

Hero Section

Changing the Headline

Find the .hero-title block in the HTML and update the three lines:

<h1 class="hero-title">
  <span class="outline">We Build</span>    ← first line (outlined)
  <span class="glitch">Digital</span>     ← middle line (gradient + glitch)
  <span class="outline">Worlds</span>      ← last line (outlined)
</h1>

Adding a Video Background

Place your video file in the same folder as index.html and update the source:

<video id="hero-video" autoplay muted loop playsinline>
  <source src="your-video.mp4" type="video/mp4">
</video>
TIP

Best results with dark, abstract, slow-moving videos (10–30 seconds, loopable). Recommended: search "dark cinematic loop" on Pexels or Mixkit for free options.

WARN

If no video file is present, the hero automatically shows a beautiful animated gradient fallback — no broken images or errors.

07 — About

About Section

Editing the Text

Find the #about section in the HTML. Update the heading, paragraphs, and stats:

<!-- Section title -->
<h2 class="section-title">We Are The <span class="accent">Future</span> Of Digital</h2>

<!-- Stats -->
<div class="stat-number">120+</div>
<div class="stat-label">Projects Completed</div>

Replacing the Visual

The right side displays an animated lava lamp visual. To replace it with your own image, find the .about-lamp-img tag and update the src:

<img class="about-lamp-img" src="your-image.jpg" alt="About">
08 — Services

Services Section

Each service card follows this structure. Copy/paste to add more, or remove cards as needed:

<div class="service-card reveal">
  <div class="service-num">01</div>
  <div class="service-name">Your Service Name</div>
  <p class="service-desc">Short description here.</p>
  <div class="service-tags">
    <span class="tag">Tag 1</span>
    <span class="tag">Tag 2</span>
  </div>
</div>
INFO

The services grid uses grid-template-columns: repeat(3, 1fr). For 2 cards use repeat(2, 1fr), for 4 cards use repeat(4, 1fr). Find .services-grid in the CSS.

09 — Portfolio

Portfolio / Work

Each work card has a unique animated canvas background. To customize a card:

<div class="work-item">
  <!-- Animation type: waves | grid | orbs | lines | dots -->
  <canvas class="work-canvas"
    data-type="waves"
    data-color1="#00f5ff"
    data-color2="#bf00ff"></canvas>
  <div class="work-overlay">
    <div class="work-cat">Category</div>
    <div class="work-title">Project Title</div>
    <div class="work-desc">Short description.</div>
  </div>
</div>

Animation Types

data-typeVisual
wavesFlowing sine waves with a glow orb
gridPerspective vanishing point grid
orbsFloating glowing orbs with connecting lines
linesDiagonal animated lines with scanlines
dotsDot grid with ripple wave distortion

11 — Particles

Particles (Three.js)

The background particle system is powered by Three.js. Find the // THREE.JS PARTICLES comment in the JS to tweak:

// Number of particles (lower = better performance)
const count = 1800;

// Particle colors (very dim — just hints of color)
const colorA = new THREE.Color('#003a3f'); // dim cyan
const colorB = new THREE.Color('#2a0015'); // dim pink
const colorC = new THREE.Color('#1a0030'); // dim purple
WARN

Keep count below 3000 for smooth performance on most devices. On mobile, 800–1000 is recommended.

12 — Animations

Animations

Scroll Reveal

Any element with the class reveal will animate in when it enters the viewport. Add it to any HTML element:

<div class="your-element reveal">...</div>

Glitch Effect

The glitch CSS animation is defined as @keyframes glitch. Apply it with:

.your-element {
  animation: glitch 4s infinite;
}

Disabling Animations

To disable all animations for accessibility or performance, add this CSS:

*, *::before, *::after {
  animation-duration: 0.01ms !important;
  transition-duration: 0.01ms !important;
}
13 — Deployment

Deployment

GitHub Pages (Free)

1

Create a repository

Go to github.com → New Repository. Name it anything (e.g. my-agency). Set it to Public.

2

Upload files

Upload index.html (and hero-bg.mp4 if you have one) to the repository.

3

Enable Pages

Go to Settings → Pages → Source: Deploy from branch → Select main → Save.

4

Your site is live

In ~2 minutes your site is live at https://yourusername.github.io/my-agency

Netlify (Recommended)

Drag and drop your folder onto netlify.com/drop — live in seconds, free SSL included.

Custom Domain

Both GitHub Pages and Netlify support custom domains. Add a CNAME record in your DNS pointing to the provided host.
